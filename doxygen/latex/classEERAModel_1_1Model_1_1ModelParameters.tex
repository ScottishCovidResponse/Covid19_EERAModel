\doxysubsection{Detailed Description}
Describes the parameters used by the model. 

Provides constants associated with the use of model parameter sets. \mbox{\hyperlink{namespaceEERAModel_1_1Model}{Model}} parameter sets are stored as vectors, where each element of the vector corresponds to a particular parameter. In order to avoid the use of lots of magic numbers in the code, this class provides index constants with meaningful names.

\begin{DoxyNote}{Note}
\mbox{\hyperlink{namespaceEERAModel_1_1Model}{Model}} parameters are distinct from \char`\"{}fixed parameters\char`\"{} to the model; the latter are also used to configure the behaviour of the model, but are not varies and inferred as part of the inference framework. The former are also referred to as \char`\"{}\+Inference Parameters\char`\"{} when speaking in the context of the inference framework. 
\end{DoxyNote}
