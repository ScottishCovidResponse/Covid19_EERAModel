void EERAModel::Inference::InferenceParticleGenerator::Update ( std::vector< particle >  particles)

Update the generator.

On each loop of the ABC-smc framework, a new collection of particles is accepted by the framework and recorded. This function should be called with the new set of particles, updating its internal weight distribution and kernel windows for the inference parameters

Parameters
particlesList of particles accepted on the last ABC-smc loop
