unsigned int  n 
)

Sum vector elements in blocks.

Summate values at every Nth index, e.g. n = 7 would summate daily data to be weekly data. For cases where n is not a multiple of the vector size, the residual values at the end of the input are dropped.

If n is greater than the size of data, or the size of data is zero, an empty vector is returned.

Parameters
dataVector of data to be reduced
nNth value at which summation should occur (i.e. index+1) where
n <= data.size()
Returns
Vector of summation values
